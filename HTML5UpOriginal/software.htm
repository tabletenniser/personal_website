PAST PROJECTS

Connect of Five

    This project was originally accomplished by me in 2009 and later modified in 2012 using Java. Connect of five or Gomoku is a simple chess game where two players place black and white stones in turn with black going first. The winner is determined to be the first player to get an unbroken row of five stones horizontally, vertically, or diagonally (check here for more information about the game). My program contains three major functionalities: enable the user to play against the computer with an attempted AI (Artificial Intelligence), allow two users to play against each other on the same computer and replay a previous game stored. An executable .jar file is available here:

Instructions on how to execute this project:
   1) Download the Java runtime environment from http://www.java.com/en/download/index.jsp if you do not have it.
   2) Download and run jarfix.exe (a freeware provided by Softpedia) from here to set up environmental variables.
   3) Extract the Connect_of_five_v1.1.zip into some editable directory and double click on the Connect_of_five_v1.1.jar to execute the program.

Image Wavelength and Video Frequency Measurement

   Those two programs utilize 2D Fourier Transform to measure most dominant wavelength of a giving pattern image and frequency of a peiodic video. They were created by Jeremy McGibbon in 2011 and later modified by me in 2012 as part of the SURF (Summer Undergraduate Research Fellowship) program in order to expedite data analysis process in vertically vibrated granular substances. Unfortunately, as a nature of the fellowship program, those projects are copyrighted by University of Toronto and cannot be uploaded here.

Game of 24

This was my final project for Grade 10 computer science course in 2009. Game of 24 is a traditional arithmetic card game in which the player manipulates simple arithmetic operations on four randomly chosen cards to make up the target number 24 (check http://en.wikipedia.org/wiki/24_Game for details). This project implements this game with a user friendly graphic user interface with functionalities like to evaluate an expression entered by the user as well as to display the correct answer for the random selected cards using a Binary Expression Tree. An executable version of this project is available from: Instructions on how to run this application on Windows:
   1) Click on the Download Game of 24 button above.
   2) Extract the .zip file into some editable location.
   3) Run Turing.exe under Turing 4.1.1 folder. (Unfortunately, Holt Software, the company that invented Turing went out of business and this language has become a freeware afterwards)
   4) Select File --> Open and find the game_of_24.t file.
   5) Click on the Run button in the Turing IDE to execute the program.
